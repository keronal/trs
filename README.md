# 🌐 TRS - 智能双语翻译

基于 **DeepSeek 大模型**的 Chrome 双语对照翻译扩展。原文译文对照显示，流畅不卡顿。

## ✨ 特性

- **双语对照** — 原文和译文对照显示，阅读体验极佳
- **DeepSeek AI 驱动** — 利用大模型的高质量翻译能力，远超传统机翻
- **13 种目标语言** — 支持简繁中文、英日韩法德西俄葡阿泰越等
- **智能过滤** — 自动跳过代码块、数字、专有名词，保留不译
- **请求队列管理** — 并发控制 + 失败重试，告别卡顿
- **内置缓存** — 相同文本不重复翻译，节省 API 额度
- **两种显示模式** — 下方显示 / 悬停提示，随心切换
- **动态内容监听** — SPA 页面新加载的内容也会自动翻译

## 📦 安装方法

### 1. 获取 DeepSeek API Key

前往 [DeepSeek 开放平台](https://platform.deepseek.com/api_keys) 注册并创建 API Key。

> 💰 **费用参考**：deepseek-chat 模型 ¥1/百万输入 tokens，¥2/百万输出 tokens。翻译一篇长文大约消耗几分钱，非常实惠。

### 2. 加载扩展

1. 打开 Chrome 浏览器，地址栏输入 `chrome://extensions/`
2. 打开右上角的 **"开发者模式"** 开关
3. 点击 **"加载已解压的扩展程序"**
4. 选择 `trs` 文件夹（即本项目根目录）
5. 扩展加载成功！

### 3. 配置 API Key

- **方式一**：点击扩展图标 → 在弹出的面板中点击 "⚠️ 请先配置 API Key" → "前往设置"
- **方式二**：右键扩展图标 → "选项"
- 在设置页填入 DeepSeek API Key，自动保存

### 4. 开始翻译

打开任意网页，点击扩展图标 → **"翻译本页"**，译文会自动出现在每段文字下方。

## 🔧 技术实现

- **Manifest V3** — 使用最新的 Chrome 扩展标准
- **Service Worker** — 后台处理 API 请求，不占用页面资源
- **请求队列** — 最多 3 个并发请求 + 指数退避重试
- **增量翻译** — 每批 8 个文本块，逐步注入，不影响页面响应
- **智能缓存** — 基于文本哈希的内存缓存（最多 2000 条）
- **MutationObserver** — 监听 DOM 变化，自动翻译动态内容
- **深度过滤** — 跳过 `<code>/<pre>/<script>` 等元素，避免翻译代码

## ❓ 常见问题

**Q: 翻译到一半卡住了？**
A: 点击扩展图标查看状态。如果是 "翻译中..."，请稍等片刻（大页面需要更多时间）。也可以点击 "停止" 后重新翻译。

**Q: 提示 "请先配置 API Key"？**
A: 右键扩展图标 → "选项"，填入 DeepSeek API Key。

**Q: 某些页面无法翻译？**
A: Chrome 内部页面（如 `chrome://`、`chrome-extension://`）和 Chrome 网上应用店不支持内容脚本注入。

**Q: 如何节省 API 额度？**
A: 扩展内置了缓存机制，相同文本不会重复翻译。你也可以在设置中调低并发数。
