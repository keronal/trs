关于 TRS

TRS - 智能双语翻译 v1.0.0

基于 DeepSeek AI 大模型的双语对照翻译扩展。


✨ 特点：

  • 原文译文对照显示，双语阅读体验
  • DeepSeek 大模型驱动，翻译质量高
  • 支持 13 种目标语言
  • 智能识别代码/数字/专有名词，保留不译
  • 请求队列管理，不卡顿、不超时
  • 内置翻译缓存，避免重复消耗 API

📝 使用说明：

  1. 在 DeepSeek 平台 获取 API Key
  2. 在本页填入 API Key 并保存
  3. 打开任意网页，点击扩展图标 → "翻译本页"

DeepSeek API 按用量计费，deepseek-v4-flash 模型价格实惠，适合日常翻译。
