# 🌐 TRS - 智能双语翻译

基于 **DeepSeek V4 大模型**的 Chrome 双语对照翻译扩展。原文译文对照显示，流畅不卡顿。

## ✨ 特性

- **双语对照** — 译文追加在原文下方，双语阅读体验极佳
- **DeepSeek V4 驱动** — 支持 DeepSeek-V4 Flash（快速）和 DeepSeek-V4 Pro（高质量）两种模型
- **13 种目标语言** — 简体中文、繁体中文、英、日、韩、法、德、西、俄、葡、阿、泰、越
- **智能过滤** — 自动跳过代码块、数字、纯符号、camelCase/snake_case 标识符、版本号、路径等
- **自适应颜色** — 译文颜色自动匹配原文色调，视觉融合自然
- **可视区域优先** — 先翻译视口内可见内容，再翻译页面其余部分
- **请求队列管理** — 可配置并发数（1~10，默认 6）+ 指数退避重试
- **持久化缓存** — 基于文本哈希的缓存（最多 2000 条），自动持久化到本地存储，相同文本不重复翻译
- **动态内容监听** — 基于 MutationObserver，SPA 页面新加载的内容自动翻译
- **自动翻译模式** — 可选开启，每次打开页面自动翻译
- **域名排除** — 可配置排除域名列表，在这些站点不触发翻译
- **快捷键** — `Alt+A` 快速开关翻译

## 📦 安装方法

### 1. 获取 DeepSeek API Key

前往 [DeepSeek 开放平台](https://platform.deepseek.com/api_keys) 注册并创建 API Key。

> 💰 **费用参考**：deepseek-v4-flash 模型 ¥1/百万输入 tokens，¥2/百万输出 tokens。翻译一篇长文大约消耗几分钱。

### 2. 加载扩展

1. 打开 Chrome 浏览器，地址栏输入 `chrome://extensions/`
2. 打开右上角的 **"开发者模式"** 开关
3. 点击 **"加载已解压的扩展程序"**
4. 选择 `trs` 文件夹（即本项目根目录）
5. 扩展加载成功！

### 3. 配置 API Key

- **方式一**：点击扩展图标 → 在弹出的面板中点击 "⚠️ 请先配置 API Key" → "前往设置"
- **方式二**：右键扩展图标 → "选项"
- 在设置页填入 DeepSeek API Key，自动保存

### 4. 开始翻译

打开任意网页，点击扩展图标 → **"翻译本页"**，译文会自动出现在每段文字下方。

## 🔧 技术实现

- **Manifest V3** — 使用最新的 Chrome 扩展标准，Service Worker 后台处理
- **请求队列** — 可配置并发数（默认 6），指数退避重试（最多 2 次），30 秒超时
- **批量翻译** — 每批 20 个文本块，批次间并行发送，由后台队列统一调度
- **智能缓存** — 内存 Map 缓存 + 定时持久化到 `chrome.storage.local`，最多 2000 条，超出时淘汰最旧 20%
- **深度过滤** — 跳过 `<script>/<style>/<code>/<pre>/<svg>/<canvas>` 等元素；识别并跳过 camelCase、snake_case、kebab-case、路径、版本号、@提及 等代码标识符
- **可视区域优先** — 按元素距视口距离排序，先翻译可见内容
- **跨批次去重** — 相同文本（忽略大小写）只翻译一次，避免浪费 API
- **自适应译文颜色** — 取父元素文字颜色与中灰混合，暗色变亮、亮色变暗，自然融入页面
- **MutationObserver** — 监听 DOM 变化，新内容出现后 1.5 秒防抖自动翻译
- **标签页关闭检测** — 翻译前检查标签页是否仍存在，已关闭的页面不浪费 API 调用
- **旧模型名迁移** — 自动将旧版 `deepseek-chat` / `deepseek-reasoner` 迁移为新版模型名

## ❓ 常见问题

**Q: 翻译到一半卡住了？**
A: 点击扩展图标查看状态。如果是 "翻译中..."，请稍等片刻（大页面需要更多时间）。也可以点击 "停止" 后重新翻译。

**Q: 提示 "请先配置 API Key"？**
A: 右键扩展图标 → "选项"，填入 DeepSeek API Key。

**Q: 某些页面无法翻译？**
A: Chrome 内部页面（如 `chrome://`、`chrome-extension://`）和 Chrome 网上应用店不支持内容脚本注入。

**Q: 如何节省 API 额度？**
A: 扩展内置了缓存机制（自动持久化），相同文本不会重复翻译。你也可以在设置中调低并发数，或在排除域名中添加不需要翻译的站点。

**Q: 如何修改快捷键？**
A: 在设置页 → 通用设置 → 点击"修改快捷键"，或在 Chrome 地址栏输入 `chrome://extensions/shortcuts` 手动配置。
